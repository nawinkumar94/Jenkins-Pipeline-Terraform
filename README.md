# Jenkins-Pipeline-Terraform
To build a infrastructure using terraform automated with jenkins.

Stages
 - Gitcheckout
 - Terraform download
 - Terraform install
 - Terraform int and validate
 - Terraform plan
 - Terraform apply/destroy

Plugin used
 - ansicolor
 - DSL plugin
 - Github

Implemented Using
 - Groovy
 - shell scripting
